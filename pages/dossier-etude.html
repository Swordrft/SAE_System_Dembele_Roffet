Dossier d'étude

Introduction

Ce projet consiste à créer et installer une machine virtuelle, puis à y configurer un serveur web Apache afin d’héberger un site internet. La machine virtuelle permet de disposer d’un environnement isolé pour l’installation et la configuration du système et des services nécessaires. Un site de documentation a également été réalisé afin de retracer les différentes étapes du projet. Celui-ci présente le parcours de création de la machine virtuelle et de l’hébergement du site à l’aide de captures d’écran, facilitant la compréhension et la reproduction des manipulations effectuées.

Création d'une machine et hébergement d'un site web via Apache

Étape 1 : Téléchargement et extraction de l'image Debian

Pour l’installation du système, nous avons téléchargé une image Debian 12 depuis le site linuxvmimages.com, qui propose des images prêtes à l’emploi pour la virtualisation. Le fichier étant compressé, nous l’avons extrait à l’aide du logiciel 7-Zip afin d’obtenir le fichier utilisable par la machine virtuelle. Debian a été choisie pour sa stabilité, sa sécurité et son utilisation fréquente dans les environnements serveurs.

Étape 2 : Installation de VirtualBox

Nous avons ensuite téléchargé VirtualBox depuis le site officiel virtualbox.org. Ce logiciel a été retenu car il est gratuit, facile à utiliser et compatible avec de nombreux systèmes d’exploitation, dont Debian. Il constitue une solution idéale pour créer et gérer des machines virtuelles dans un cadre pédagogique.

Étape 3 : Ouverture de la machine virtuelle dans VirtualBox

Nous avons ouvert la machine virtuelle dans VirtualBox en cliquant sur le bouton « Nouvelle ». Nous avons ensuite choisi le type de machine virtuelle « Linux » et l'image Debian 12 . Nous avons ensuite cliqué sur « Suivant » et « Terminer » pour créer la machine virtuelle.

Étape 4 : Configuration de la machine virtuelle

Ensuite nous avons verifié que la machine virtuelle était bien configurée en allant dans les paramètres de la machine virtuelle et en vérifiant que le système d'exploitation était bien Debian 12. Et avons ensuite lancé la machine virtuelle.

Étape 5 : Premier démarrage de Debian

Apres avoir lancé la machine virtuelle, nous avons vu l'écran de connexion de Debian. Nous avons ensuite entré le mot de passe « debian » pour accéder à la machine virtuelle.

Étape 6 : Écran d'accueil Debian

Apres avoir entré le mots de passes, nous somme arrivé sr l'ecran de configuration de la machine ou nosu avons choisi la langue"englaisé et le clavier "français azerti" et nousavons ensuite cliqué sur "continuer".

Étape 7 : Installation d'Apache2

Via la commande "sudo apt-get install apache2" nous avons installé le serveur web Apache2.

Étape 8 : Configuration des dossiers partagés

Ensuite nous avons configuré les dossiers partagés en allant dans les paramètres de la machine virtuelle "peripheriques", "dossier partagés" et ensuite nous avons choissi le dossier reliant la machine avec l'ordinateur ici "vm dossier partagé".

Création du site

Le site web a été créé afin de présenter et documenter l’ensemble du projet. Il a été réalisé en HTML, avec une mise en forme exclusivement assurée par Bootstrap, permettant d’obtenir un design clair, responsive et cohérent sans utiliser de CSS personnalisé. Le choix de limiter le site à trois pages permet une navigation simple et efficace. La page Accueil présente le contexte du projet ainsi que le diagramme de Gantt, offrant une vue globale de l’organisation et des objectifs. La page Virtualisation explique les différents types de virtualisation utilisés dans une machine virtuelle, afin d’apporter les bases théoriques nécessaires. Enfin, la page Dossier d’étude regroupe l’ensemble des tâches réalisées, illustrées par des captures d’écran, afin de documenter clairement chaque étape du projet.

Conclusion du dossier

La réalisation de ce projet nous a permis de mettre en pratique la création et la configuration d’une machine virtuelle ainsi que l’hébergement d’un site web via Apache. Le choix de VirtualBox s’est imposé grâce à sa gratuité, sa simplicité d’utilisation et sa compatibilité avec de nombreux systèmes d’exploitation, ce qui en fait un outil idéal pour l’apprentissage et les environnements de test. L’utilisation d’une image Debian a été retenue pour sa stabilité, sa fiabilité et sa légèreté. Debian est une distribution largement utilisée dans le monde des serveurs, offrant une grande sécurité et une gestion efficace des paquets, ce qui la rend particulièrement adaptée à l’hébergement de services web.
