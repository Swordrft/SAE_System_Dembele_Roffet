Virtualisation

Définition de la virtualisation

La virtualisation est une technologie que vous pouvez utiliser pour créer des représentations virtuelles de serveurs, de stockage, de réseaux et d'autres machines physiques. Le logiciel virtuel imite les fonctions du matériel physique pour exécuter plusieurs machines virtuelles sur une seule machine physique. Les entreprises ont recours à la virtualisation pour utiliser efficacement leurs ressources matérielles et obtenir un meilleur rendement de leurs investissements. Elle alimente également les services de cloud computing qui aident les organisations à gérer leur infrastructure plus efficacement.

Différences entre les outils

Émulateur

Un émulateur est un logiciel qui reproduit entièrement le fonctionnement d’un autre système, y compris son matériel. Il permet par exemple d’exécuter un système ou un programme prévu pour une autre architecture que celle de l’ordinateur utilisé.

Simulateur

Un simulateur est un outil qui imite le comportement d’un système sans en reproduire exactement le fonctionnement réel. Il sert principalement à tester ou à observer des situations, sans chercher à exécuter réellement le système comme sur une vraie machine.

Outil de virtualisation

Un outil de virtualisation permet de créer et d'exécuter des machines virtuelles sur un ordinateur physique. Ces machines utilisent le matériel réel de l'ordinateur hôte et fonctionnent comme de vrais systèmes, tout en restant isolées les unes des autres.

Types de virtualisation

VirtualBox

VirtualBox utilise une virtualisation de type 2. Le logiciel de virtualisation est installé comme une application classique. Il permet de créer et exécuter plusieurs machines virtuelles sur un même ordinateur physique. Ce type de virtualisation est surtout utilisé pour l’apprentissage, les tests et le développement.

VMware Workstation

VMware Workstation utilise également une virtualisation de type 2. Comme VirtualBox, il dépend d’un système d’exploitation hôte. Il offre généralement de meilleures performances et plus de fonctionnalités avancées, comme la gestion réseau ou les snapshots. Il est très utilisé par les professionnels, développeurs et administrateurs systèmes pour tester des environnements.

Stratus

Stratus utilise une virtualisation de type 1 (bare metal). L’hyperviseur est installé directement sur le matériel, sans système d’exploitation intermédiaire. Ce type de virtualisation offre une meilleure performance, une haute disponibilité et une forte tolérance aux pannes. Stratus est principalement utilisé dans des environnements critiques (banques, industries, hôpitaux) où l’arrêt des systèmes est inacceptable.
